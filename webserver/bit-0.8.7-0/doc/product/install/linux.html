Installing on Linux

The Embedthis Bit distribution for Linux is the Bit source distribution, available here.

Building and Installing Bit

  1. Download and extract the source code.
  2. Change directory to the top level of the source code .
  3. Do a bootstrap make via "make boot".
  4. Install via "make install"".
tar xfz ~/Downloads/bit-0.8.0-0-src.tgz
cd bit-0.8.0
make boot
sudo make install

Removing Bit

  1. Run a terminal and change directory to the installation directory for Bit. This is typically "/usr/lib/bit/latest".
  2. Run the uninstall script in the Bit install directory: 
    cd /usr/lib/bit/latest
sudo ./uninstall

Running Bit

You can invoke Bit simply as bit 

$ bit
bit.es: ERROR: Can't find suitable start.bit.
Run "bit --gen start" to create stub start.bit

This indicates you are ready to start by creating your first start.bit file. To learn more, read the Bit Overview.
