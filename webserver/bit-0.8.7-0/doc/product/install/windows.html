Installing on Windows

The Bit install package for Windows is an installation executable compressed in a ZIP file available here.

The distribution image contains the binary distribution and documentation headers in the one package.

Installing the Image

  1. Login with administrator privileges.
  2. Run winzip and extract the contents of the zipped EXE install image to a folder of your choosing.
  3. Browse to the location holding the EXE install image and run it.

Removing Bit

  1. Login with administrator privileges
  2. Run the Add / Remove Programs applet from the control panel.
  3. Select Embedthis Bit and choose remove.

Silent Installation

If you need to do batch or unattended installations, you can install Bit silently. Use the /silent switch to suppress questions and /verysilent to suppress all visual output.

Running Bit

You can invoke Bit simply as bit 

$ bit
bit.es: ERROR: Can't find suitable start.bit.
Run "bit --gen start" to create stub start.bit

This indicates you are ready to start by creating your first start.bit file. To learn more, read the Bit Overview.
