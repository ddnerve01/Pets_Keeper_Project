Bit Roadmap

Bit is currently in pre-release. We are still actively developing and changing how it works. So there will be incompatible, breaking changes, while we move toward a 1.0 release.

The Bit roadmap is created by listening to user input. As such, it is subject to change. This roadmap is not a commitment to product, but the current thinking about future directions.

Our current priorities are

  • Stabilize recent features
  • Improve scalability by remembering C include dependencies
  • Improve consistency of when Bit propertys have ${tokens} expanded
  • Launch the 3rd party package repository. This will allow the easy exchange of Bit package extensions.

Remember, those who shout loudest -- often get what they ask for. To request a feature, please send email to: dev@embedthis.com
