Building with an IDE

Bit provides Visual Studio and Xcode projects. These projects are provide easy access to debug Bit using the native system debugger. The projects are somewhat limited in their configurability.

Building with Visual Studio on Windows

To build Bit with the Visual Studio project, run Visual Studio 2010 or later (The Express editions are fine to use). Then open the projects/bit-windows.sln solution project. Select Build Solution to build. Select Bit as the start up project to debug.

Building with Xcode on MAC OS X

To build Bit with the Xcode on Mac OSX, run Xcode and open the projects/bit-macosx.xcodeproj project. Then select Product->Build to build. To debug, you will need to create use Product->Edit Scheme to set the executable to run under the Info tab, and define the working directory under the Options tab. Set the working directory to the "test" directory of the source tree.
