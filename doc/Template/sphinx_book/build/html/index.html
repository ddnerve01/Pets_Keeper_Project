用Sphinx写书¶

写技术书是一件十分费时费力的事情，作者不但需要编写有趣的内容，还需要用标准且美观大方的格式呈现内容。在编写《Python科学计算》一书的过程中，我尝试使用Sphinx、Leo、MiKTeX等软件，拼凑出了一套适合编写技术书籍和文档的编写环境。这本书是关于这个编写环境的一些介绍。
